第 5 課

字串

5.1 字串 = 字元陣列

C 的字串是「字元陣列」，結尾有一個看不見的空字元 \0。用索引 s[i] 取單一字元，<string.h> 的 strlen 取長度：

C
#include <string.h>
char s[100] = "hello";
printf("%c\n", s[0]);            // h   用索引取單一字元
printf("%zu\n", strlen(s));      // 5   字串長度

5.2 讀入一整行

要讀入可能含空白的一整行，用 fgets，再把結尾的換行去掉：

C
char s[1000];
fgets(s, sizeof s, stdin);       // 讀入一整行（可含空白）
s[strcspn(s, "\n")] = 0;         // 去掉結尾換行

5.3 逐字元走訪

用迴圈跑過每個字元（到 \0 為止）：

C
int i;
for (i = 0; s[i] != '\0'; i++)   // 走訪每個字元，字串以 \0 結尾
    putchar(s[i]);
putchar('\n');

5.4 反轉印出

從最後一個索引往前印，就能反轉：

C
int i, L = strlen(s);
for (i = L - 1; i >= 0; i--)     // 從最後往前印 = 反轉
    putchar(s[i]);
putchar('\n');

5.5 常用格式

%c 印單一字元、%s 印整個字串、%.5s 印前 5 個字元、%.*s 可用變數指定長度；scanf("%s", w) 讀入一個不含空白的單字。
