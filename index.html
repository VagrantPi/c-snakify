第 1 課

第一支程式、輸出與輸入

1.1 你的第一支 C 程式

C 程式從 main 函式開始執行。下面這支程式會在螢幕印出一行字：

C
#include <stdio.h>

int main() {
    printf("Hello, world!\n");
    return 0;
}
  • #include <stdio.h>：引入標準輸入輸出函式庫，才能使用 printf / scanf。
  • int main() { ... }：程式進入點，大括號內是要執行的敘述。
  • printf("...")：把字串印到螢幕，\n 代表換行。
  • return 0;：告訴系統程式正常結束。
  • 每一行敘述都要以分號 ; 結尾。

編譯與執行（終端機）：

C
cc hello.c -o hello   # 編譯，產生執行檔 hello
./hello               # 執行
輸出
Hello, world!

1.2 註解

註解是給人看的說明，編譯器會忽略：

C
// 這是單行註解
/* 這是
   多行註解 */

1.3 變數與整數

變數要先「宣告型別」再使用。整數用 int，用 printf 的 %d 印出：

C
#include <stdio.h>

int main() {
    int age = 18;
    printf("%d\n", age);   // 印出 18
    return 0;
}

%d 是「佔位符」，會被後面的變數值取代。

1.4 讀取輸入：scanf

用 scanf 從鍵盤讀入資料。注意變數前要加 &（取位址）：

C
#include <stdio.h>

int main() {
    int n;
    scanf("%d", &n);       // 讀入一個整數存進 n
    printf("%d\n", n);     // 原樣印出
    return 0;
}

一次讀多個，用空白分隔格式字串：

C
int a, b;
scanf("%d %d", &a, &b);
printf("%d\n", a + b);   // 印出兩數之和

第 2 課

資料型別與運算

2.1 常用資料型別

型別用途printf 格式範例
int整數%dint n = 10;
float / double小數（浮點數）%fdouble x = 3.14;
char單一字元%cchar g = 'A';

2.2 算術運算子

+ 加、- 減、* 乘、/ 除、% 取餘數。乘除餘優先於加減，可用括號改變順序。

C
int a = 17, b = 5;
printf("%d\n", a + b);   // 22
printf("%d\n", a / b);   // 3  （整數除法，小數被捨去）
printf("%d\n", a % b);   // 2  （餘數）

2.3 整數除法 vs 浮點除法（重要陷阱）

兩個整數相除，結果會「捨去小數」。想得到小數，其中一邊要是浮點數：

C
printf("%d\n", 7 / 2);        // 3   （整數除法）
printf("%.1f\n", 7 / 2.0);    // 3.5 （有一邊是 2.0 → 浮點除法）

%.1f 表示印出小數點後 1 位。

2.4 型別轉換（cast）

用 (型別) 可臨時把值轉成另一種型別：

C
int a = 7, b = 2;
printf("%.2f\n", (double)a / b);  // 3.50

2.5 取出位數的技巧

% 10 取個位數，/ 10 去掉個位數：

C
int n = 456;
printf("%d\n", n % 10);   // 6   個位數
printf("%d\n", n / 10);   // 45  去掉個位

第 3 課

條件：if / else

3.1 if：條件成立才執行

C
int n = 5;
if (n > 0) {
    printf("positive\n");
}

括號內的條件為「真」（非 0）時，才執行大括號內的敘述。

3.2 if / else

C
if (n % 2 == 0) {
    printf("even\n");
} else {
    printf("odd\n");
}

3.3 比較運算子

== 等於、!= 不等於、> 大於、< 小於、>=、<=。注意：判斷相等是兩個等號 ==，一個等號 = 是「指派」。

3.4 else if 連鎖

C
if (n > 0) {
    printf("positive\n");
} else if (n < 0) {
    printf("negative\n");
} else {
    printf("zero\n");
}

3.5 邏輯運算子

&&（且，兩者皆真）、||（或，其一為真）、!（非）。

C
int score = 85;
if (score >= 60 && score < 70) {
    printf("D\n");
}

L3-5成績等第

讀入一個分數（0–100），依 >=90 A、>=80 B、>=70 C、>=60 D、其餘 F 印出等第。

範例輸入
85
範例輸出
B
參考解答（已編譯執行驗證 ✓）
C
#include <stdio.h>
int main() {
    int s;
    scanf("%d", &s);
    char g;
    if (s >= 90)      g = 'A';
    else if (s >= 80) g = 'B';
    else if (s >= 70) g = 'C';
    else if (s >= 60) g = 'D';
    else              g = 'F';
    printf("%c\n", g);
    return 0;
}
