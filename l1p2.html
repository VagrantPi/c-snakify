L1-2 練習題

Hi John

讀入一個名字，印出 Hi 加上這個名字（中間一個空格）。

測試項目

你的程式要在下列每組輸入都印出對應的預期輸出。

輸入預期輸出
John
Hi John
Alice
Hi Alice
Bob
Hi Bob
Tanner
Hi Tanner
Jake
Hi Jake

本地測試指令

① 用你的 IDE 把解答存成 solution.c。② 打開同一個資料夾的終端機。③ 複製下面整段貼上執行 —— 它會自動編譯並用上面所有測試項目檢查，回報通過/失敗。
cc solution.c -o sol || exit
p=0; f=0
t(){ g=$(printf '%s\n' "$1" | ./sol); if [ "$g" = "$2" ]; then echo "✓ $1"; p=$((p+1)); else printf '✗ 輸入[%s] 期望[%s] 得到[%s]\n' "$1" "$2" "$g"; f=$((f+1)); fi; }
t 'John' 'Hi John'
t 'Alice' 'Hi Alice'
t 'Bob' 'Hi Bob'
t 'Tanner' 'Hi Tanner'
t 'Jake' 'Hi Jake'
echo "== 通過 $p / 失敗 $f =="

需要 C 編譯器（macOS/Linux 內建 cc；Windows 可用 WSL 或 MinGW 的 gcc，把指令的 cc 換成 gcc）。
