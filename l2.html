第 2 課

整數與浮點數

2.1 用 % 和 / 取出各位數字

n % 10 取個位；n / 10 % 10 取十位；n / 100 % 10 取百位：

C
int n = 2024;
printf("%d\n", n % 10);       // 4   個位
printf("%d\n", n / 10 % 10);  // 2   十位
printf("%d\n", n / 100 % 10); // 0   百位

2.2 各位數字之和

把每一位取出來相加：

C
int n = 123;
printf("%d\n", n / 100 + n / 10 % 10 + n % 10);   // 6  三位數各位之和

2.3 浮點數的讀取與輸出

浮點數用 double；scanf 讀入要用 %lf，printf 印出用 %f（%.4f = 小數點後 4 位）：

C
double x;
scanf("%lf", &x);       // 讀浮點數要用 %lf
printf("%.4f\n", x);     // 印小數點後 4 位

2.4 取小數部分與小數位

減掉整數部分得到小數部分；乘 10 取整再 % 10 取得小數第一位：

C
double x = 3.75;
printf("%.4f\n", x - (long long)x);   // 0.7500  小數部分
printf("%d\n", (int)(x * 10) % 10);    // 7       小數點後第一位

2.5 無條件進位（整數技巧）

「不足一份也算一份」可用 (m + n - 1) / n 純整數算出：

C
int m = 250, n = 100;   // 每天 n 公里，走 m 公里
printf("%d\n", (m + n - 1) / n);   // 3  （無條件進位）

2.6 時間換算

分鐘數換算成「時 分」：

C
int t = 150;   // 從午夜起的分鐘數
printf("%d %d\n", t / 60 % 24, t % 60);   // 2 30
