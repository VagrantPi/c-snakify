資料型別與運算

2.1 常用資料型別

型別用途printf 格式範例
int整數%dint n = 10;
float / double小數（浮點數）%fdouble x = 3.14;
char單一字元%cchar g = 'A';

2.2 算術運算子

+ - * / %：乘除餘優先於加減，可用括號改變順序。

C
int a = 17, b = 5;
printf("%d\n", a + b);   // 22
printf("%d\n", a / b);   // 3  （整數除法，小數被捨去）
printf("%d\n", a % b);   // 2  （餘數）

2.3 整數除法 vs 浮點除法（重要陷阱）

兩個整數相除會「捨去小數」。想得到小數，其中一邊要是浮點數：

C
printf("%d\n", 7 / 2);        // 3   （整數除法）
printf("%.1f\n", 7 / 2.0);    // 3.5 （有一邊是 2.0 → 浮點除法）

%.1f 表示印出小數點後 1 位。

2.4 型別轉換（cast）

用 (型別) 把值臨時轉成另一種型別：

C
int a = 7, b = 2;
printf("%.2f\n", (double)a / b);  // 3.50

2.5 取出位數的技巧

% 10 取個位數，/ 10 去掉個位數：

C
int n = 456;
printf("%d\n", n % 10);   // 6   個位數
printf("%d\n", n / 10);   // 45  去掉個位
