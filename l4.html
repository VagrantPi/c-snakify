第 4 課

for 迴圈

4.1 for 迴圈

for (初始化; 條件; 每次遞增)：只要條件成立就重複執行大括號內的敘述。

C
int i;
for (i = 0; i < 5; i++) {
    printf("%d ", i);   // 0 1 2 3 4
}
printf("\n");

4.2 從 a 數到 b

改變起始值與條件，就能從任意範圍開始（i++ 遞增、i-- 遞減）：

C
int i;
for (i = 3; i <= 7; i++) {   // 從 3 到 7
    printf("%d ", i);        // 3 4 5 6 7
}
printf("\n");

4.3 累加

迴圈外先設一個變數、迴圈內不斷累加，是最常見的模式：

C
int s = 0, i;
for (i = 1; i <= 100; i++) {
    s += i;   // 累加
}
printf("%d\n", s);   // 5050

4.4 讀入 N 個數字

先讀數量 N，再用迴圈逐一 scanf 讀入並處理：

C
int n, x, s = 0, i;
scanf("%d", &n);            // 先讀數量 N
for (i = 0; i < n; i++) {
    scanf("%d", &x);        // 迴圈中逐一讀入
    s += x;
}
printf("%d\n", s);

4.5 巢狀迴圈

迴圈裡再放迴圈，可印出階梯／表格等圖形：

C
int i, j;
for (i = 1; i <= 3; i++) {       // 外層：每一列
    for (j = 1; j <= i; j++) {   // 內層：印 1..i
        printf("%d", j);
    }
    printf("\n");
}
