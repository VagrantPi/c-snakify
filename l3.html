第 3 課

條件：if / else

3.1 if：條件成立才執行

C
int n = 5;
if (n > 0) {
    printf("positive\n");
}

括號內的條件為「真」（非 0）時，才執行大括號內的敘述。

3.2 if / else

C
if (n % 2 == 0) {
    printf("even\n");
} else {
    printf("odd\n");
}

3.3 比較運算子

== 等於、!= 不等於、>、<、>=、<=。注意：判斷相等是兩個等號 ==，一個等號 = 是「指派」。

3.4 else if 連鎖

C
if (n > 0) {
    printf("positive\n");
} else if (n < 0) {
    printf("negative\n");
} else {
    printf("zero\n");
}

3.5 邏輯運算子

&&（且）、||（或）、!（非）。可用來組合條件，例如判斷範圍或多個座標關係。

C
int score = 85;
if (score >= 60 && score < 70) {
    printf("D\n");
}

3.6 絕對值 abs()

取兩數差距的絕對值，引入 <stdlib.h> 後用 abs()（棋子移動、距離判斷常用）：

C
#include <stdlib.h>   // 用 abs() 需引入
int d = abs(3 - 8);   // 5   兩數差距（絕對值）

3.7 三元運算子（一行版 if/else）

條件 ? A : B：成立取 A、否則取 B，常用來簡潔印出 YES/NO：

C
int a = 3, b = 8;
printf("%d\n", a < b ? a : b);            // 3
printf("%s\n", (a == b) ? "YES" : "NO");   // NO
