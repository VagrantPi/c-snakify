第 3 課

條件：if / else

3.1 if：條件成立才執行

C
int n = 5;
if (n > 0) {
    printf("positive\n");
}

括號內的條件為「真」（非 0）時，才執行大括號內的敘述。

3.2 if / else

C
if (n % 2 == 0) {
    printf("even\n");
} else {
    printf("odd\n");
}

3.3 比較運算子

== 等於、!= 不等於、>、<、>=、<=。注意：判斷相等是兩個等號 ==，一個等號 = 是「指派」。

3.4 else if 連鎖

C
if (n > 0) {
    printf("positive\n");
} else if (n < 0) {
    printf("negative\n");
} else {
    printf("zero\n");
}

3.5 邏輯運算子

&&（且）、||（或）、!（非）。

C
int score = 85;
if (score >= 60 && score < 70) {
    printf("D\n");
}
