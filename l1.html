第 1 課

輸入、輸出與數字

1.1 你的第一支 C 程式

C 程式從 main 函式開始執行。下面這支程式會在螢幕印出一行字：

C
#include <stdio.h>

int main() {
    printf("Hello, world!\n");
    return 0;
}
  • #include <stdio.h>：引入標準輸入輸出函式庫，才能使用 printf / scanf。
  • int main() { ... }：程式進入點，大括號內是要執行的敘述。
  • printf("...")：把字串印到螢幕，\n 代表換行。
  • return 0;：告訴系統程式正常結束。
  • 每一行敘述都要以分號 ; 結尾。

編譯與執行（終端機）：

C
cc hello.c -o hello   # 編譯，產生執行檔 hello
./hello               # 執行
輸出
Hello, world!

1.2 註解

註解是給人看的說明，編譯器會忽略：

C
// 這是單行註解

   多行註解 */

1.3 變數與整數

變數要先「宣告型別」再使用。整數用 int，用 printf 的 %d 印出：

C
#include <stdio.h>

int main() {
    int age = 18;
    printf("%d\n", age);   // 印出 18
    return 0;
}

1.4 讀取數字輸入：scanf

用 scanf 從鍵盤讀入資料。注意變數前要加 &（取位址）。一次讀多個，用空白分隔格式字串（換行也可）：

C
#include <stdio.h>

int main() {
    int n;
    scanf("%d", &n);       // 讀入一個整數存進 n
    printf("%d\n", n);     // 原樣印出
    return 0;
}
C
int a, b;
scanf("%d %d", &a, &b);
printf("%d\n", a + b);   // 印出兩數之和

1.5 讀取與印出文字（字串）

字串用「字元陣列」char name[100] 存放，%s 讀入一個不含空白的字、也用 %s 印出（讀字串時變數不用加 &）：

C
#include <stdio.h>

int main() {
    char name[100];
    scanf("%99s", name);       // 讀入一個字（不含空白）
    printf("Hi %s\n", name);   // %s 印出字串
    return 0;
}

1.6 算術運算、整數除法與餘數

+ - * /。兩個整數相除是「整數除法」會捨去小數；% 取餘數：

C
int a = 17, b = 5;
printf("%d\n", a + b);   // 22
printf("%d\n", a / b);   // 3  （整數除法，小數被捨去）
printf("%d\n", a % b);   // 2  （餘數）

1.7 浮點數除法

想得到小數，其中一邊要是浮點數（例如 / 2.0）；%.1f 印出小數點後 1 位：

C
int a = 3, b = 4;
printf("%.1f\n", a * b / 2.0);  // 6.0  （有一邊是 2.0 → 浮點除法）
