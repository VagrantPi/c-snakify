第 1 課

第一支程式、輸出與輸入

1.1 你的第一支 C 程式

C 程式從 main 函式開始執行。下面這支程式會在螢幕印出一行字：

C
#include <stdio.h>

int main() {
    printf("Hello, world!\n");
    return 0;
}
  • #include <stdio.h>：引入標準輸入輸出函式庫，才能使用 printf / scanf。
  • int main() { ... }：程式進入點，大括號內是要執行的敘述。
  • printf("...")：把字串印到螢幕，\n 代表換行。
  • return 0;：告訴系統程式正常結束。
  • 每一行敘述都要以分號 ; 結尾。

編譯與執行（終端機）：

C
cc hello.c -o hello   # 編譯，產生執行檔 hello
./hello               # 執行
輸出
Hello, world!

1.3 變數與整數

變數要先「宣告型別」再使用。整數用 int，用 printf 的 %d 印出：

C
#include <stdio.h>

int main() {
    int age = 18;
    printf("%d\n", age);   // 印出 18
    return 0;
}

1.4 讀取輸入：scanf

用 scanf 從鍵盤讀入資料。注意變數前要加 &（取位址）：

C
#include <stdio.h>

int main() {
    int n;
    scanf("%d", &n);       // 讀入一個整數存進 n
    printf("%d\n", n);     // 原樣印出
    return 0;
}

一次讀多個，用空白分隔格式字串：

C
int a, b;
scanf("%d %d", &a, &b);
printf("%d\n", a + b);   // 印出兩數之和
